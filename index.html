Project Name:

Notebook LM Project

Project Summary:

The organization is facing challenges with high AHT, low FCR, and increased defects, leading to poor customer experience, higher operational costs, and increased FTE consults. By improving these metrics, we aim to enhance efficiency, reduce costs, and improve customer satisfaction.

Project Goal:

Project goal was to Reduce AHT by 20% within 6 months, Improve first-call resolution (FCR) by 15%, Reduce knowledge search errors by 25% and Increase agent satisfaction with KB usage by 30%

Project Tools Used:

Python

SQL

Power BI

Six Sigma DMAIC

Output:

Reduced inventory costs by 15% and improved order fulfillment time by 20%.
